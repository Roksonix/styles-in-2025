# Text

Text is a typography component that can be used to render text with different styles.

By passing fontVariant, you can change the font size and weight of the text.

Replacement for direct usage of setFont/setFontVariant in CSS-in-JS code.

## Usage

```jsx
import { Text } from '@similarweb/ui-components/foundations';

...

<Flex>
  <Text fontVariant="input2">Input variant</Text>
</Flex>
```
